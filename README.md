# Goodreads Books Data Analysis

## Objective
To analyze Goodreads book data and uncover trends in ratings, popularity, and book characteristics.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Key Insights
- Most books are rated between 3.5 and 4.5
- Popular books receive disproportionately more text reviews
- Book length has little impact on average rating
- A small number of authors contribute multiple books

## Dataset
Goodreads Books Dataset (Kaggle)
